Sobre

Mesa Posta

é quando você monta todos os utensílios para uma refeição, como pratos, talheres, copos e adornos de uma maneira organizada e estética. A principal função é deixar o ambiente agradável e bonito para receber convidados. A mesa posta está muito atrelada às regras de etiqueta.

É importante lembrar que a ideia da mesa posta é distribuir de uma maneira padronizada tudo o que você precisa para uma refeição – cada utensílio tem uma função específica. Assim, a mesa posta é também adaptável de acordo com o menu. Por exemplo, se você não vai servir peixe, obviamente o garfo para esse prato não precisa estar posto. A ideia é facilitar para você que estará anfitriando não precisar correr pra lá e pra cá com louças durante o evento, além de deixar o local mais agradável para os convidados.
